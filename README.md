# AI Notes Summarizer

An AI-powered web application that converts long study notes into short, clear, and meaningful summaries using Natural Language Processing.

# Features
- Summarizes long study notes instantly
- Simple and clean web interface
- Built using Hugging Face Transformers
- Runs fully in the browser using Streamlit
- Useful for students and self-learners

# Tech Stack
- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
